# 2.3 Requisitos Funcionais

Os requisitos funcionais descrevem os serviços, funcionalidades e comportamentos que o sistema AgiliCar deve disponibilizar para atender às necessidades identificadas durante o levantamento dos processos de negócio.

Esses requisitos foram definidos a partir das histórias de usuário e representam as capacidades essenciais da solução para apoiar as atividades de recepcionistas, técnicos e gestores da oficina.

## Requisitos Funcionais

| ID     | Descrição do Requisito                                                                           | Prioridade |
| ------ | ------------------------------------------------------------------------------------------------ | ---------- |
| RF-001 | Permitir autenticação de usuários com e-mail e senha                                             | ALTA       |
| RF-002 | Controlar o acesso por perfil: Gestor, Recepcionista e Técnico                                   | ALTA       |
| RF-003 | Buscar veículo cadastrado pela placa para identificação na entrada                               | ALTA       |
| RF-004 | Cadastrar novo cliente e veículo                                                                 | ALTA       |
| RF-005 | Realizar checklist visual do veículo com registro de observações e fotos                         | ALTA       |
| RF-006 | Registrar os sintomas relatados pelo cliente e abrir uma Ordem de Serviço (OS)                   | ALTA       |
| RF-007 | Listar e visualizar as OS recebidas com filtro por status                                        | ALTA       |
| RF-008 | Registrar o diagnóstico técnico da OS                                                            | ALTA       |
| RF-009 | Adicionar peças e serviços à OS a partir do catálogo cadastrado                                  | ALTA       |
| RF-010 | Gerar o orçamento da OS com o valor total das peças e serviços                                   | ALTA       |
| RF-011 | Enviar o orçamento ao cliente e registrar sua decisão (aprovado ou reprovado)                    | ALTA       |
| RF-012 | Atribuir um técnico responsável à OS após aprovação do orçamento                                 | ALTA       |
| RF-013 | Registrar a execução do serviço com fotos e anotações do técnico                                 | ALTA       |
| RF-014 | Registrar a entrega do veículo ao cliente com quilometragem e confirmação de entrega             | ALTA       |
| RF-015 | Gerenciar o cadastro de funcionários (criar, editar, desativar)                                  | MÉDIA      |
| RF-016 | Gerenciar o cadastro de peças com quantidade em estoque e estoque mínimo                         | MÉDIA      |
| RF-017 | Gerenciar o catálogo de serviços prestados pela oficina                                          | MÉDIA      |
| RF-018 | Exibir relatórios operacionais com indicadores de OS, status e produtividade                     | MÉDIA      |
| RF-019 | Registrar os pagamentos realizados vinculados à OS, com valor, data, forma de pagamento e status | MÉDIA      |
| RF-020 | Gerar relatórios financeiros e operacionais com receitas, pagamentos, peças e serviços           | MÉDIA      |
